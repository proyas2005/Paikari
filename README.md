# Paikari

community powered price intelligency network

🛒 Paikari. Community Price Intelligence Network

Paikari is a web platform that helps buyers and sellers in Bangladesh know the prices of products. It lets users find, compare and check prices before buying.


📌 Overview

In markets in Bangladesh product prices are different depending on where you are which shop you go to and what time it is. Paikari solves this problem by creating a platform where users and businesses can share and compare prices.

Features

 Verified Price Submissions

Users can submit product prices with receipts or photos to prove they are real.

 🏪 Business Accounts

Shops and wholesalers can publish their prices and build trust.

 📊 Live Price Comparison

Compare prices across different shops, brands and locations instantly.

 📈 Price History & Trends

See how prices change over time to know if they are fair or not.

 👍 Community Voting System

Users can vote to say if a price is correct or not.

📍 Location-Based Insights

Compare prices in areas like Dhaka and Chattogram.


👥 User Roles

 General Users

They can search, compare submit prices and vote on if they are accurate.

- Business Accounts

They can publish their prices and keep a good reputation.

- Admins/Moderators

They manage users look at flagged data and make sure the data is good.


🛠️ Technology Stack

- Laravel. A PHP framework for the backend

- MySQL. For managing the database

-  Bootstrap. For designing the UI

- gihub and cloud for devOps

- Git and GitHub. For version control



⚙️ How It Works

1. Submit. Users upload product prices with proof.

2. Verify. The community and businesses check the submissions.

3. Vote. Users say if a price is correct or not.

4. Publish. Trusted prices are shown.

5. Compare. Users easily. Compare prices.


⚠️ Challenges

- Making submitted data is real

- Getting many users to join and be active

- Stopping spam or cheating

- Keeping location data clean and organized
